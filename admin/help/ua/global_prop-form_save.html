Створення архівної копії сайту


Для того, щоб зберегти архівну копію сайту, Вам необхідно відмітити ті частини сайту, які Вам необхідно зберігати (файли контента - те, що редагується через візуальний редактор розділу "Структура", Таблиці БД (якщо у Вас є магазин чи стрічка новин тощо.) та/або системні файли)

Відновлення архівної копії сайту

Переконайтесь, що Ви встановили права на запис 777 для всього сайту. Сделать это можно в разделе "Системные действия" кликнув по ссылке "Установить полные права на файлы"
