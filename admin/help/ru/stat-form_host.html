Отчет посещаемости сайта

Этот отчет покажет Вам посещаемость Вашего сайта.

Хиты - количество просмотренных страниц.

Хосты - количество посетителей

Отношение Хиты/Хосты показывает - сколько страниц в среднем просматривает каждый посетитель. Чем это отношение больше- тем лучше аудитория сайта.

Показывать только людей - учитывются только те посещения, в которых отработал специальный яваскрипт. Это позволяет отсечь посещения индексирующих роботов поисковых систем и прочих программных средств.
