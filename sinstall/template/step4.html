Введённые параметры

FTP хост: [#ftp_value_hostadress#]   Хост сервера MySql: [#mysql_value_hostadress#]
FTP логин: [#ftp_value_login#]   Имя базы данных: [#mysql_value_namebase#]
FTP пароль: [#ftp_value_password#]   Логин к MySql: [#mysql_value_login#]
Путь инсталяции: [#ftp_value_path#]   Пароль к MySql: [#mysql_value_password#]
админ логин: [#admin_login#]   Тип инсталяции: [#install_type#]
админ пароль: [#admin_pass#]   Кодировка шаблонов [#templates_encoding#]

Если вся информация введена верно, то сейчас будет создан файл ini.php
